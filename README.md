# <devkaua/> - Portfólio Pessoal 🚀

Bem-vindo ao repositório do meu portfólio pessoal! Este projeto foi desenvolvido inicialmente como uma atividade prática para a faculdade (Engenharia de Software na Uninter), mas foi projetado e estruturado para ser meu portfólio real e definitivo na área de tecnologia.

Você pode acessar o projeto online aqui: [https://devkauapimentel.github.io/portfolio-kaua/](https://devkauapimentel.github.io/portfolio-kaua/)

## 💻 Sobre o Projeto

O objetivo principal desse portfólio é demonstrar minhas habilidades com **Front-End Vanilla**. Eu tomei a decisão de **não utilizar nenhum framework** (como React, Angular ou TailwindCSS) para mostrar que tenho domínio sólido das bases da web: estruturação semântica, estilização avançada e manipulação direta da DOM com JavaScript.

Aqui você encontra um pouco sobre minha trajetória acadêmica, meus hobbies (que incluem xadrez e teologia), as tecnologias que estou focado em aprender e, claro, meus projetos de destaque.

## ✨ Funcionalidades (Features)

- **Suporte a Múltiplos Idiomas (i18n):** Alternância em tempo real entre Português (PT) e Inglês (EN) usando JavaScript nativo.
- **Dark/Light Mode:** Troca de tema fluida com salvamento da preferência do usuário no `localStorage`.
- **Animações "Glass Reveal":** Elementos aparecem suavemente ao rolar a página com um efeito combinado de desfoque (`blur`), opacidade e escala, criando uma sensação de profundidade.
- **Scroll Spy:** A barra de navegação rastreia a posição atual do usuário na página e destaca o link ativo da seção correspondente.
- **Responsividade Pura:** Layout construído com Flexbox e CSS Grid, adaptável para celulares, tablets e desktops (Mobile-First).
- **Validação de Formulário:** O formulário de contato possui validação em tempo real no Front-End para garantir o preenchimento correto dos dados.

## 🛠️ Tecnologias Utilizadas

Este projeto foi construído usando as tecnologias web fundamentais:

- **HTML5:** Semântico e acessível.
- **CSS3:** Animações, CSS Grid, Flexbox e variáveis nativas (Custom Properties) para gerenciar o tema.
- **JavaScript (Vanilla):** Toda a lógica de interatividade, idioma, tema e scroll feita do zero sem bibliotecas externas.

## 🚀 Como Rodar o Projeto Localmente

Como o projeto não utiliza ferramentas de build ou dependências como o Node.js, rodar localmente é extremamente simples:

1. Faça o clone do repositório:
```bash
git clone https://github.com/devkauapimentel/portfolio-kaua.git
```
2. Abra a pasta do projeto no seu editor de código favorito (recomendo o VS Code).
3. Abra o arquivo `index.html` diretamente no seu navegador, ou utilize a extensão **Live Server** para habilitar o auto-reload durante edições.
